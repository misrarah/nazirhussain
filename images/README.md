## Tips

- **Format:** JPEG (`.jpg`) is recommended for photos. Keep file sizes reasonable — aim for under 500 KB per image for fast loading.
- **Compression:** Use a tool like [Squoosh](https://squoosh.app/) or [TinyJPG](https://tinyjpg.com/) to compress before uploading.
- **Orientation:** Landscape photos (wider than tall) work best for the gallery grid. The hero and portrait are handled with `object-fit: cover` so any aspect ratio will work.
- **Rights:** Ensure you own or have permission to use all images published on the site.
